# NBA API Scraper

API simple que hace web scraping de estadísticas NBA y las expone en un endpoint.

## Cómo usar

1. Instalar dependencias:
   npm install

2. Ejecutar scraper manual:
   npm run scrape

3. Ejecutar servidor:
   npm start

## Endpoint

GET /data

## Deploy

Subir a GitHub y conectar con Render.
